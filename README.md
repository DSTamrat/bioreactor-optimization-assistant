\# AI-Generated Bioreactor Optimization Assistant



This project simulates an upstream bioreactor process and builds:



1\. A model to recommend feed rates.

2\. A dashboard to monitor bioreactor health.

3\. Plain-English anomaly explanations.



\## Main components



\- Synthetic dataset (1,000 rows)

\- ML model (RandomForestRegressor)

\- Anomaly detection logic

\- Streamlit dashboard
